## styled components

```
// 기본 styled-components
yarn add styled-components @types/styled-components

// 문자열 안에 스타일 들어가는 것 처리를 위한 설치
yarn add -dev babel-plugin-styled-components

// 전역 스타일링에서 이용하기 위함
yarn add styled-reset
```
